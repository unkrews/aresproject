# Ares Command Center

Ares Command Center — це інтерактивний вебсимулятор управління марсіанською колонією, створений спеціально для конкурсу LogiRace 2026 (Web Creators). Проєкт імітує футуристичну операційну систему, де гравець виступає в ролі Командора, відповідального за життя поселенців у ворожому середовищі.

🛠 Технологічний стек
Проєкт розроблений без використання сторонніх бібліотек (Vanilla JS), щоб продемонструвати глибоке розуміння базових технологій:

HTML5: Семантична розмітка інтерфейсу.

CSS3: Використання Grid Layout для адаптивної сітки дашборду, Custom Properties (змінних) для керування станами та анімацій keyframes для сповіщень про небезпеку.

JavaScript (ES6+): Об'єктно-орієнтований підхід до ігрової логіки, маніпуляції з DOM та робота з таймерами.

Web Audio API: Створення процедурного звукового середовища та звукових ефектів у реальному часі.

🌌 Основні можливості
1. Система керування ресурсами (State Management)
Проєкт відстежує чотири критичні показники:

Кисень (O₂): Постійно вичерпується; потребує витрат енергії для регенерації.

Енергія (⚡): Базовий ресурс, що відновлюється сонячними панелями, але витрачається іншими системами.

Їжа (🍞): Необхідна для підтримки життєдіяльності колонії.

Населення (👨‍🚀): Статичний показник успіху місії.

2. Процедурна генерація подій
Вбудований ігровий двигун випадковим чином генерує виклики для гравця:

Піщані бурі (втрата енергії).

Метеоритні дощі (пошкодження систем кисню).

Космічна радіація та приємні бонуси у вигляді знайдених ресурсів.

3. Динамічний UI та Audio-занурення
Smart Indicators: Смуги прогресу змінюють колір (синій → помаранчевий → червоний) залежно від критичності рівня ресурсу.

Event Log: Кожна подія та дія гравця фіксується у системному журналі з міткою часу.

Sound Engine: Включає фоновий ембієнт та окремі тональності для успішних дій, попереджень та критичних станів.
